## ðŸ”„ Archive Process

When a file needs to be archived:

1. **Move file** to this folder
2. **Add date suffix:** `filename_YYYY-MM-DD.md`
3. **Update CHANGELOG.md** with:

   - Why it was archived
   - What replaced it (if applicable)
4. **Update references** in other documents
5. **Keep for reference** (don't delete)
